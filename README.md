# Smart-Bluetooth-Braille-Keyboard
Smart Bluetooth-Enabled Braille Keyboard is a portable assistive device for visually impaired users. Built using ESP32, Embedded C++, and BLE HID, it converts 6-dot Braille key combinations into digital text and transmits them wirelessly to smartphones, tablets, and computers in real time.

## Features
- 6-dot Braille chording system
- ESP32 microcontroller
- Bluetooth Low Energy (BLE) HID communication
- Lowercase, uppercase, number, and special character modes
- Rechargeable Li-ion battery
- 3D-printed enclosure

## Hardware Used
- ESP32
- Push Buttons
- Li-ion Battery (3.7V)
- TP4056 Charging Module
- MT3608 Boost Converter

## Software Used
- Arduino IDE
- Embedded C++
- ESP32 BLE Keyboard Library

## Applications
- Assistive technology
- Accessible text input
- Digital communication for visually impaired users
